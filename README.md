# img-flow
My linux based workflow for photos


*Rsync files from laptop/mobile to server collector.
*Import files from collector to archive.
*Prune collector.
*Prune laptop.
*Backup archive.
*Upload to Google Photo.
*Acessable with file sharing.

Prereq:
perl-Image-ExifTool
rsync
fdupes

Filetypes:
nef
jpg
mov
